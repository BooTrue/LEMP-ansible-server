Параметры сервера указываются в файле /vars/default.yml

1) В папке configs хранятся файлы с конфигурациями для nginx (domain.ru.conf). 
Сертификат прописывается в верхний сервер конфига на 443 порту, ниже расположен конфиг для 80 порта.

2) Стандартный вид переменной list_subdomens - list_subdomens: ''. Это означает, что псевдонимов к домену нет, 
необходимо указывать для корректной генерации SLL сертификата. Если необходимо указать псевдонимы к домену,
то это будет выглядеть:

list_subdomens:
    - www.example.ru
    - dev.example.ru
